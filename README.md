# InfiniAttention PT-BR - Implementation of InfiniAttention in PT-BR with GPT-2 model

The gpt2 with Infini from HuggingFace transformers it's in "03_gpt2_infini.ipynb" notebook.
The InfiniTransformer it's implemented in "03_infini_transformer.ipynb" notebook.

The last notebook it's the final implementation and it's not integrated at all with HuggingFace Transformers (because of some extra interfaces needed to add, futher work).

To run the notebooks you need to generate some dataset with "02_data_modeling.ipynb" and change use the "%cd _PATH_WITH_DATA_" in the notebook "03_infini_transformer.ipynb".
